This is a toy language made for learning. The interpreter utilizes newlines to seperate statements

Variable declaration:


  • let {variable_name} = {value}
  • Example: let i = 10
Reassignment to variables also require the usage of let keyword.
  • Example: let i = 10
    let i = i + 1

Printing:


print {string or number or identifier}
  • Example: print "hello world"
  • print 1+2
  • print i

if condition:


The interpreter only supports if condition and no else keyword
if {condition} then
statement
endif
Example:
if i < 10 then
print i
endif

while loop:


while {condition} repeat
statement
endwhile
Example:
while i < 10 repeat
print i
i = i+1
endwhile
